Сборка:
Запустить скрипт script.sh, предварительно сделав его исполняемым, в директории hw5 (той же, где находится файл pom.xml).

Очистка:
Выполнить команду mvn clean.

Запуск: 
Выполнить команду:
    java -cp ./target/hw5-1.0-SNAPSHOT.jar ru.spbau.group202.fl.sharkova.hw5.Main %полный путь к файлу%
(выполнять из директории hw5)
jar-файл находится в созданной директории target.
Результат выводится в терминал.
